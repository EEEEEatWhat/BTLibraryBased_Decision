## 简介
基于BehaviorTree.CPP(v4.4)和BehaviorTree.ROS2的行为树，为机器人提供智能人工决策设计

吉林大学TARS_Go战队2024赛季 哨兵决策代码（不完整版）。

## 代码框架

```sh
├── README.md                            // 项目自述文件
└── src
    ├── bt_test                          // 仅供节点测试使用
    |   
    ├── fake_msg_publisher               // 发布测试消息
    |   
    ├── decision                         // 决策功能包
    |
    ├── global_interfaces                //全局通用消息文件
    |
    ├── lib                              // 主要依赖库
    |    ├── BehaviorTree.CPP
    |    └── BehaviorTree.ROS2
```
